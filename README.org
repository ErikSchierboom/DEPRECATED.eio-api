** Usage

*** Run the application locally

=lein ring server=

*** Run the tests

=lein midje=

*** Packaging and running as standalone jar

#+BEGIN_SRC Clojure
lein do clean, ring uberjar
java -jar target/server.jar
#+END_SRC

*** Packaging as war

=lein ring uberwar=
